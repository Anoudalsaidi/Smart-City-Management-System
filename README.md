# 🌆 Smart City Management System

A modern Smart City Management System API built with ASP.NET Core, Entity Framework Core, SQL Server, JWT Authentication, and Clean Architecture.

- Entity Framework Core
- SQL Server
- JWT Authentication
- Clean Architecture
- Repository Pattern
- AutoMapper
- FluentValidation
- Swagger
- Serilog
- Docker
- GitHub Actions
